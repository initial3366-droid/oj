# QOJ 项目目录重构建议

## 当前目录结构

```
qoj/
├── backend/                    # 后端 Spring Boot 项目
│   ├── src/
│   ├── pom.xml
│   └── target/                 # ⚠️ 构建产物，应该在 .gitignore
├── src/                        # ⚠️ 前端源码在根目录
│   ├── components/
│   ├── pages/
│   └── data/
├── node_modules/              # ⚠️ 前端依赖，应该在 .gitignore
├── dist/                      # ⚠️ 前端构建产物，应该在 .gitignore
├── .runtime/                  # Docker Compose 配置
├── package.json               # 前端依赖配置
├── vite.config.ts             # 前端构建配置
├── .env                       # ⚠️ 敏感配置，应该在 .gitignore
├── qoj_database_backup.sql    # ⚠️ 数据库备份，应该在 .gitignore
└── .DS_Store                  # ⚠️ macOS 临时文件，应该在 .gitignore
```

**问题分析**：
1. ❌ 前端代码直接放在根目录，与后端混在一起
2. ❌ 构建产物（dist, target, node_modules）与源码混在一起
3. ❌ 敏感文件（.env）和临时文件（.DS_Store, *.sql）未排除
4. ❌ 缺少统一的文档目录
5. ❌ 基础设施配置（Docker Compose）位置不规范

## 安全注意事项

⚠️  **重要**: 生产环境部署前请务必：

1. 修改 `JWT_SECRET` 为强密码（至少 64 字节）
2. 修改所有默认密码
3. 禁用 `LocalJudgeService`（存在安全风险）
4. 配置 DOMjudge 进行安全判题
5. 启用 HTTPS
6. 配置防火墙

### 3. 创建部署脚本

```bash
mkdir -p scripts

cat > scripts/setup.sh << 'EOF'
#!/bin/bash
# QOJ 项目初始化脚本

set -e

echo "🚀 QOJ 项目初始化..."

# 检查依赖
command -v node >/dev/null 2>&1 || { echo "❌ Node.js 未安装"; exit 1; }
command -v java >/dev/null 2>&1 || { echo "❌ Java 未安装"; exit 1; }
command -v mvn >/dev/null 2>&1 || { echo "❌ Maven 未安装"; exit 1; }
command -v docker >/dev/null 2>&1 || { echo "❌ Docker 未安装"; exit 1; }

echo "✅ 依赖检查通过"

# 复制环境变量
if [ ! -f .env ]; then
    cp .env.example .env
    echo "✅ 已创建 .env 文件，请修改其中的配置"
else
    echo "⚠️  .env 文件已存在，跳过"
fi

# 启动数据库和 Redis
echo "🐳 启动 Docker 容器..."
docker compose -f .runtime/qoj-deps.compose.yml up -d

# 等待数据库就绪
echo "⏳ 等待数据库就绪..."
sleep 10

# 安装前端依赖
echo "📦 安装前端依赖..."
npm install

# 构建后端
echo "🔨 构建后端..."
cd backend
mvn clean compile
cd ..

echo "✅ 初始化完成！"
echo ""
echo "下一步："
echo "1. 修改 .env 中的配置"
echo "2. 启动后端: cd backend && mvn spring-boot:run"
echo "3. 启动前端: npm run dev"
echo "4. 访问: http://localhost:5173"
EOF

chmod +x scripts/setup.sh
```

## 目录重构的实施步骤

### 方案 A：渐进式重构（推荐）

**第4周**: 前端独立

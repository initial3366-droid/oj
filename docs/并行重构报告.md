### 2. Controller使用Policy

```java
@PutMapping("/{id}")
public ApiResponse<Void> update(@PathVariable Long id, @RequestBody ProblemUpdateRequest request) {
    Problem problem = problemMapper.selectById(id);
    if (problem == null) {
        throw new BizException(ErrorCode.NOT_FOUND);
    }
    
    AuthUser user = CurrentUser.required();
    if (!problemAccessPolicy.can(user, Permission.UPDATE, problem)) {
        throw new BizException(ErrorCode.FORBIDDEN);
    }
    
    problemService.update(id, request);
    return ApiResponse.ok();
}
```

### 3. Service使用Policy

```java
public ProblemVO detail(long id) {

        throw new BizException(ErrorCode.NOT_FOUND);
    }
    
    return toVO(problem);
}
```

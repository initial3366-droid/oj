## 发现的安全问题

### 1. **严重漏洞：Practice 提交验证缺失** ⚠️

**问题描述**:
- `SubmissionService.submit()` 接受 `practiceId` 参数，但没有任何验证
- 用户可以提交任意题目到任意练习
- 没有检查 practice 是否存在、是否可访问、problem 是否属于 practice

**攻击场景**:
```bash
POST /api/v1/submissions
{
  "problemId": 999,        # 任意题目
  "practiceId": 123,       # 任意练习
  "code": "...",
  "language": "cpp"
}
# 请求成功，数据库记录 practice_id=123，但该题目根本不属于练习
```

**影响**:
- 练习榜单数据错误（统计了不属于练习的提交）
- 权限绕过（可以通过私有练习提交公开题目）
- 数据完整性破坏

### 2. **中等漏洞：Practice 列表信息泄露**

**问题描述**:
- `PracticeService.list()` 返回所有 `published=true` 的练习
- 班级练习和社团练习也出现在公开列表中
- 用户可以看到不应该看到的练习信息

**攻击场景**:
```bash
GET /api/v1/practices
# 返回所有已发布练习，包括：
# - 某班级的内部练习
# - 某社团的成员练习
# - 设置了密码的私密练习
```

### 3. **轻度缺陷：ensureVisible() 未登录检查不完整**

**问题描述**:
- `ensureVisible()` 中 `CurrentUser.required()` 对于公开练习会抛出 401
- 未登录用户无法访问公开练习（`audience=ALL`）

### 4. **功能缺失：软删除未实现**

**问题描述**:
- Practice 表没有 `is_deleted` 和 `deleted_at` 字段
- 删除操作是物理删除（永久丢失数据）
- 无法恢复误删除的练习

#### 1.3 添加验证方法

```java
/**
 * 验证 Practice 提交的合法性
 */
private void validatePracticeSubmission(SubmissionCreateRequest request, AuthUser user) {
    // 1. 验证 practice 是否存在
    Practice practice = practiceMapper.selectById(request.practiceId());
    if (practice == null) {
        throw new BizException(ErrorCode.NOT_FOUND, "练习不存在");
    }

    // 2. 验证 practice 是否已发布
    if (!Boolean.TRUE.equals(practice.published)) {
        throw new BizException(ErrorCode.FORBIDDEN, "练习未发布");
    }

    // 3. 验证用户是否有提交权限（使用 Policy）
    if (!practiceAccessPolicy.can(user, Permission.SUBMIT, practice)) {
        throw new BizException(ErrorCode.FORBIDDEN, "无权提交该练习");
    }

    // 4. 验证 audience 权限（班级/社团成员检查）
    if (!"ALL".equals(practice.audience)) {
        validatePracticeAudience(practice, user);
    }

    // 5. 验证 problem 是否属于 practice
    Long problemInPracticeCount = practiceProblemMapper.selectCount(
        new QueryWrapper<PracticeProblem>()
            .eq("practice_id", request.practiceId())
            .eq("problem_id", request.problemId())
    );
    if (problemInPracticeCount == null || problemInPracticeCount == 0) {
        throw new BizException(ErrorCode.BAD_REQUEST, "该题目不属于该练习");
    }
}

/**
 * 验证 Practice 的 audience 权限
 */
private void validatePracticeAudience(Practice practice, AuthUser user) {
    // 超级管理员和创建者跳过检查
    if ("SUPER_ADMIN".equals(user.role()) || practice.ownerId.equals(user.id())) {
        return;
    }

    if ("CLASS".equals(practice.audience) && practice.audienceId != null) {
        Long memberCount = classMemberMapper.selectCount(
            new QueryWrapper<ClassMember>()
                .eq("class_id", practice.audienceId)
                .eq("user_id", user.id())
        );
        if (memberCount == null || memberCount == 0) {
            throw new BizException(ErrorCode.FORBIDDEN, "该练习仅限指定班级成员");
        }
    } else if ("CLUB".equals(practice.audience) && practice.audienceId != null) {
        Long memberCount = clubMemberMapper.selectCount(
            new QueryWrapper<ClubMember>()
                .eq("club_id", practice.audienceId)
                .eq("user_id", user.id())
        );
        if (memberCount == null || memberCount == 0) {
            throw new BizException(ErrorCode.FORBIDDEN, "该练习仅限指定社团成员");
        }
    }
}
```

**验证逻辑**:
1. ✅ Practice 存在性检查
2. ✅ Practice 发布状态检查
3. ✅ 用户提交权限检查（Policy）
4. ✅ Audience 成员资格检查（班级/社团）
5. ✅ Problem 属于 Practice 检查

### 修复 3: 重构 ensureVisible() 权限检查

**修改文件**: `PracticeService.java`

#### 修改后

```java
private void ensureVisible(Practice practice) {
    AuthUser user = CurrentUser.get();  // ✅ 允许未登录用户访问公开练习

    if (!practiceAccessPolicy.can(user, Permission.VIEW, practice)) {
        // Policy拒绝后，检查是否是audience限制的问题
        if (!AudienceType.ALL.name().equals(practice.audience)) {
            // ✅ 必须登录才能检查班级/社团成员
            if (user == null) {
                throw new BizException(ErrorCode.UNAUTHORIZED.getCode(), "请先登录");
            }

            // 检查班级成员
            if (AudienceType.CLASS.name().equals(practice.audience) && practice.audienceId != null) {
                Long memberCount = classMemberMapper.selectCount(
                    new QueryWrapper<ClassMember>()
                        .eq("class_id", practice.audienceId)
                        .eq("user_id", user.id())
                );
                if (memberCount != null && memberCount > 0) {
                    return;
                }
                throw new BizException(ErrorCode.FORBIDDEN.getCode(), "该练习仅限指定班级成员");
            }

            // 检查社团成员
            if (AudienceType.CLUB.name().equals(practice.audience) && practice.audienceId != null) {
                Long memberCount = clubMemberMapper.selectCount(
                    new QueryWrapper<ClubMember>()
                        .eq("club_id", practice.audienceId)
                        .eq("user_id", user.id())
                );
                if (memberCount != null && memberCount > 0) {
                    return;
                }
                throw new BizException(ErrorCode.FORBIDDEN.getCode(), "该练习仅限指定社团成员");
            }
        }
        throw new BizException(ErrorCode.FORBIDDEN.getCode(), "无权访问该练习");
    }
}
```

**修复效果**:
- ✅ 未登录用户可以访问公开练习
- ✅ 班级/社团成员检查正确执行
- ✅ 错误消息更加明确

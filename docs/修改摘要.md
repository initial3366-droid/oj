### 4. 权限控制加固 ✅

## API 端点清单

### 排名查询（公开访问）
- `GET /api/contests/{id}/rank/acm` - 获取 ACM 榜单
- `GET /api/contests/{id}/rank/oi` - 获取 OI 榜单
- `GET /api/contests/{id}/scoreboard/snapshot/{type}` - 获取快照

### 排名管理（需要管理员权限）
- `POST /api/contests/{id}/rank/acm/rebuild` - 重建 ACM 榜单
- `POST /api/contests/{id}/rank/oi/rebuild` - 重建 OI 榜单
- `POST /api/contests/{id}/scoreboard/snapshot?type={freeze|final}` - 创建快照
- `DELETE /api/contests/{id}/scoreboard/snapshot/{type}` - 删除快照

## 核心功能验证

### 排名系统
- ✅ ACM 模式：解题数、罚时、最后 AC 时间排序
- ✅ OI 模式：总分、满分题数、最后得分时间排序
- ✅ 幂等性：重复判题不会重复加分/罚时
- ✅ 事务性：判题回调保证原子性
- ✅ 可重建：可从 submissions 表全量重建

- ✅ 公开路径：匿名用户可查看榜单
- ✅ 管理路径：需要 SUPER_ADMIN/TEACHER/CLUB_ADMIN 角色
- ✅ JWT 认证：所有非公开 API 需要登录

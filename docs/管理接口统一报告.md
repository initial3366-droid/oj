## 📦 新增文件清单

### 1. **AdminApi.java**
**路径**: `backend/src/main/java/com/qoj/security/annotation/AdminApi.java`

**功能**: 管理端权限注解

**使用示例**:
```java
@AdminApi(
    resource = ResourceType.PROBLEM,
    action = Action.UPDATE,
    requireOwner = true
)
@PutMapping("/{id}")
public ApiResponse<ProblemVO> update(@PathVariable Long id) {
    // 自动检查：用户是否登录、是否管理员、是否拥有该题目
}
```

**支持的资源类型**:
- `NONE` - 无需资源权限检查
- `PROBLEM` - 题目
- `CONTEST` - 比赛
- `PRACTICE` - 练习
- `USER` - 用户
- `ORGANIZATION` - 组织（班级/社团）
- `SUBMISSION` - 提交
- `ANNOUNCEMENT` - 公告

**支持的操作类型**:
- `READ` - 读取
- `CREATE` - 创建
- `UPDATE` - 更新
- `DELETE` - 删除

---

### 2. **AdminApiInterceptor.java**
**路径**: `backend/src/main/java/com/qoj/security/AdminApiInterceptor.java`

**功能**: 管理端权限拦截器，实现三层检查

**权限检查流程**:
```
1. 检查用户是否登录 → 未登录返回 401
2. 检查用户是否有管理员角色 → 无角色返回 403
3. 检查用户是否有资源权限 → 无权限返回 403
   - 超级管理员跳过资源权限检查
   - requireOwner=true 时检查资源所有权
```

**资源所有权检查**:
- 题目：检查 `ownerId` 是否匹配
- 比赛：检查 `ownerId` 和 `ownerAccountType`
- 练习：检查 `ownerId` 是否匹配

---

### 3. **WebMvcConfig.java**
**路径**: `backend/src/main/java/com/qoj/config/WebMvcConfig.java`

**功能**: 注册管理端拦截器

```java
@Override
public void addInterceptors(InterceptorRegistry registry) {
    registry.addInterceptor(adminApiInterceptor)
        .addPathPatterns("/api/admin/v1/**");
}
```

### 4. **AdminSubmissionController.java**

## 🔐 权限机制

### 三层权限检查

**第一层：SecurityConfig (URL 级别)**
```java
.requestMatchers("/api/admin/v1/users/**").hasRole("SUPER_ADMIN")
.requestMatchers("/api/admin/v1/**").hasAnyRole("SUPER_ADMIN", "TEACHER", "CLUB_ADMIN")
```

**第二层：@PreAuthorize (Controller 级别)**
```java
@PreAuthorize("hasAnyRole('SUPER_ADMIN','TEACHER','CLUB_ADMIN')")
public class AdminProblemController {
```

**第三层：AdminApiInterceptor + @AdminApi (资源级别)**

### 权限测试清单

**未登录用户**:
- [ ] 访问任意管理端接口 → 应返回 401

**普通用户**:
- [ ] 访问任意管理端接口 → 应返回 403

**TEACHER 角色**:
- [ ] 访问用户管理接口 → 应返回 403
- [ ] 访问自己创建的题目 → 应返回 200
- [ ] 访问他人创建的题目 → 应返回 403（非 SUPER_ADMIN）
- [ ] 访问仪表盘 → 应返回 200

**SUPER_ADMIN 角色**:
- [ ] 访问所有管理端接口 → 应返回 200
- [ ] 访问所有资源 → 应返回 200
